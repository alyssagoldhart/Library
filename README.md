# Library

Objective: Display a library of books that the user can edit.

Skills: Intermediate HTML, CSS and Javascript.

Requirements:
- Create book objects with an object constructor
- Functions
    - Store book objects in an array
    - Display books
- Buttons
    - New book
      - Allows user to input book details
      - Adds new book to the library
    - Delete book
      - Remove book from the library
    - Change read status
      - Toggle a book instance's read status and update the display
